# 🌻 小葵瓜幸存者 (Little KuiGua Survivors)

一款类吸血鬼幸存者风格的弹幕射击 + Roguelike 游戏。控制可爱的猫咪角色，在僵尸潮中生存下去！

## 🎮 游戏玩法

### 基本操作
- **移动**: WASD / 方向键 / 虚拟摇杆（移动端）
- **攻击**: 自动攻击最近的敌人
- **切换武器**: 数字键 1-3
- **背包**: ESC 键 / 点击背包按钮
### 游戏目标
- 击杀敌人获取经验和金币
- 升级时选择祝福强化角色
- 收集技能掉落，搭配法杖构建流派
- 击败Boss获取强力武器
- 尽可能存活更长时间！

## 🐱 可选角色

### 瓜瓜 (Guagua)
- **外观**: 布偶猫，棕白色毛发，蓝色眼睛
- **初始祝福**: 疾风（移动速度+15%）
- **特点**: 灵活机动，适合走位流

### 葵葵 (Kuikui)  
- **外观**: 蓝白英短，戴红色项圈和金色铃铛，橙色眼睛
- **初始祝福**: 生命强化（最大生命+20）
- **特点**: 更加耐打，适合站撸流

## 🪄 武器系统

### 武器属性
| 属性 | 说明 |
|------|------|
| 能量上限 | 法杖的最大能量值 |
| 能量回复 | 每秒回复的能量 |
| 施法间隔 | 两次施法之间的帧数 |
| 技能槽 | 可装备的技能数量 |
| 特殊槽 | 满足条件触发的额外技能槽 |

## ✨ 主动技能

主动技能装备在法杖槽位中，自动向敌人发射。

| 技能 | 图标 | 伤害 | 能量 | 说明 |
|------|------|------|------|------|
| 火花弹 | ✨ | 6 | 1 | 快速的小型魔法弹 |
| 魔法箭 | ➤ | 10 | 1 | 精准高速的箭矢 |
| 火球术 | 🔥 | 18 | 2 | 燃烧的火球，带尾迹 |
| 回旋镖 | 🪃 | 15×2 | 2 | 飞出后返回，去回双伤 |
| 飞剑 | 🗡️ | 15 | 2 | 挥舞飞剑攻击前方，可抵挡弹道 |
| 魔导弹 | 🚀 | 20 | 3 | 追踪敌人的导弹 |
| 能量球 | 💠 | 25 | 3 | 缓慢但强力，穿透5个敌人 |
| 黑洞 | 🕳️ | 8/tick | 8 | 吸附并持续伤害敌人 |

### 技能星级
技能可以通过合成升星（3个同星级 → 1个高星级）：
- ⭐ 1星: 基础效果
- ⭐⭐ 2星: 效果×1.5
- ⭐⭐⭐ 3星: 效果×2.5

## 🎯 游戏技巧

### 新手建议
1. 优先升级**伤害提升**和**攻击速度**祝福
2. 保持移动，不要站在原地
3. 注意Boss的攻击预警（红色区域）
4. 合理搭配主动技能和被动技能

### 进阶技巧
1. **穿透+散射**组合可以大幅提升清怪效率
2. **黑洞**配合**伤害增幅**可以快速击杀Boss
3. **献祭**祝福6级左右可以覆盖几乎全屏
4. 利用**拓展**技能增加槽位，但注意伤害惩罚
5. **傀儡娃娃**可以让副武器自动攻击

### 流派推荐
- **站撸流**: 葵葵 + 生命强化 + 吸血 + 献祭
- **走位流**: 瓜瓜 + 疾风 + 暴击 + 伤害提升
- **黑洞流**: 黑洞 + 伤害增幅 + 膨胀 + 穿透
- **弹幕流**: 散射 + 弹射 + 闪电链
